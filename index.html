Система 089

Уникальная тернарная система Galavox использует цифры 0, 8 и 9, где 0=0, 8=1, 9=2.
Пример: 10₁₀ = 808₀₈₉

Расширенная поддержка

Более 40 систем счисления с историческими названиями: от Binary до Base-95

Универсальные символы

Base 2–36: 0–9, A–Z
Base 37–62: + a–z
Base 63–95: расширенный printable ASCII
